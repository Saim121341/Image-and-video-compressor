<h1 align="center">Advanced Image and Video Compressor</h1>


<h2>Description</h2>
<p>Welcome to the Advanced Image and Video Compressor - a versatile Python tool designed for efficiently reducing the file sizes of images and videos without significantly compromising on quality. It supports multiple formats including JPEG, PNG, MKV, and MP4, and integrates cutting-edge compression techniques.</p>

<h2>Key Features</h2>
<ul>
  <li>Versatile Image Compression: Optimize JPEG and PNG files with adjustable settings.</li>
  <li>Robust Video Compression: Compress MKV and MP4 files using powerful libraries.</li>
  <li>User-Friendly Interface: Simple prompts guide through the compression process.</li>
  <li>Batch Processing: Automated batch compression for handling multiple files efficiently.</li>
</ul>

<h2>Getting Started</h2>

<h3>Prerequisites</h3>
<p>Before installing, ensure the following are set up on your system:</p>
<ul>
  <li>Python: <a href="https://www.python.org/downloads/">Download Python</a></li>
  <li>Pillow: Install via pip with <code>pip install Pillow</code></li>
  <li>OpenCV-Python: Install via pip with <code>pip install opencv-python</code></li>
  <li>FFmpeg-Python: Install via pip with <code>pip install ffmpeg-python</code></li>
</ul>

<h3>Installation</h3>
<p>Clone the repository and navigate to the folder:</p>

<pre><code>git clone https://github.com/Saim121341/Image-and-video-compressor.git
cd Image-and-video-compressor</code></pre>

<h3>Running the Application</h3>
<p>Execute the script and follow the interactive prompts:</p>

<pre><code>python compressor.py</code></pre>

<h2>How it Works</h2>
<ul>
  <li><code>compress_img</code>: Compresses images, with options for format conversion and size adjustment.</li>
  <li><code>compress_videos</code>: Compresses videos, tailored for MKV and MP4 formats.</li>
  <li><code>main</code>: Coordinates the program flow and user interactions.</li>
</ul>
